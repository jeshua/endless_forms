Power Flow

Battery + → Switch → 1H → 6H (5V Bus) → Splits:
  • 6A → 9A (XIAO 5V power)
  • 6C → 20C (LED ring 5V)
Battery - → 2H → 7H (GND Bus) → Splits:
  • 7B → 9B (XIAO GND)
  • 7A → 17A (10K resistor to GND)
  • 7D → 20D (LED ring GND)
XIAO 3V3 Output → 9C → 20A (FSR power)

Signal Connections

FSR Sensor (Voltage Divider):
  • FSR + wire → 20A pad (3.3V from 9C)
  • FSR - wire → 20B pad → 16A junction
  • Junction 16A connects:
    • 15A (D0) reads voltage
    • 17A (10K to GND)
LED Ring:
  • LED 5V wire → 20C pad (from 5V bus)
  • LED GND wire → 20D pad (from GND bus)
  • LED DIN wire → 20E pad → 15B (D1)

Critical Traces (Back of Board)

Power Buses (horizontal wires):
  • 6A-6H: 5V bus (solder wire across all holes)
  • 7A-7H: GND bus (solder wire across all holes)
Bus to XIAO connections:
  • 6A → 9A: 5V bus to XIAO (crosses over row 7 GND bus)
  • 7B → 9B: GND bus to XIAO (direct 2-row trace)
Component connections:
  • 15A → 16A: D0 to FSR junction (1 row)
  • 17A → 7A: 10K resistor to GND bus
  • 9C → 20A: 3V3 to FSR+ pad
  • 15B → 20E: D1 to LED DIN pad
  • 6C → 20C: 5V bus to LED 5V pad
  • 7D → 20D: GND bus to LED GND pad
  • 1H → 6H: Battery+ to 5V bus
  • 2H → 7H: Battery- to GND bus
